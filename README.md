# Builder Calculator v1.2.1

Приложение для строителей с минималистичным дизайном. Помогает быстро рассчитывать объемы бетона, количество плитки и конвертировать единицы измерений.

## Основные функции

- **Избранное** - сохраняйте расчеты для быстрого доступа:
  - Все данные хранятся в браузере (работает без интернета)
  - Подробная информация о каждом расчете
  - Удаление отдельных расчетов или очистка всего списка
  - Визуальные индикаторы количества сохраненных расчетов

- **Конвертер единиц измерений** - мгновенно переводите:
  - Длину: метры ↔ сантиметры ↔ миллиметры ↔ дюймы ↔ футы
  - Объем: кубометры ↔ литры ↔ см³
  - Вес: килограммы ↔ тонны ↔ граммы
  - Удобные подсказки по конвертации
  - Быстрые шаблоны для частых операций

- Калькулятор бетона с расчетом объема и веса
- Калькулятор плитки для пола и стен
- Поддержка темной/светлой темы
- Адаптивный дизайн для мобильных устройств
